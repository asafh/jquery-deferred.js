CallbackList Class

Defined in: deferred.js\deferred.js:12
Module: deferred

Utility object to contain a list of callbacks that can be fired with arguments and a context.
Callback lists can be configured to delete listeners upon file (i.e. 'once'), and/or to retain last fired arguments and context to send to newly add listeners (i.e. 'memory')
