Defined in deferred.js\deferred.js:152

If state is defined, attempts to change to it, firing any listeners upon change.
Trying to change from a final state has no effect, doesn't any event listeners and returns false.

Parameters:

  • state String

    the state name to change into. undefined has no affect and just retrieves the state.

  • context Any

    the context to send to the event listeners.

  • arguments any...

    the arguments to pass forward to the event listeners

Returns:

false if trying to change a final state, else the state after the method invocation
